# 🧭 ¿Cómo selecciono un servicio?

Este paso te permite elegir qué tipo de servicio querés cotizar o reservar dentro del flujo de **Reservas Online**.  
Desde acá vas a poder acceder a las secciones de búsqueda, tarifas, carrito y nominación.

![Seleccionar solapa](/img/reservas-online/seleccionar-solapa.gif)

---

## ✨ Tipos de servicios disponibles

Cada servicio tiene sus propios campos y particularidades.  
Elegí uno para ingresar directamente a su formulario:

- 🏨 [Alojamiento](../alojamiento/alojamiento.md)  
- 🚐 [Traslados](../traslados/traslados.md)  
- 🌄 [Excursiones](../excursiones/excursiones.md)  
- 🩺 [Asistencias](../asistencias/asistencias.md)

:::tip
> 💡 Si necesitás cambiar el cliente, arriba a la izquierda está el acceso directo:  
> **[« Cambiar cliente »](./seleccionar-cliente.md)**
:::
---

## 🔎 Próximos pasos del flujo

Una vez elegido el servicio, el recorrido continúa así:

1. **[Búsqueda y Resultados](../general/filtros-y-resultados.md)** — para ver las opciones disponibles.  
2. **[Ver tarifas y detalle](../general/ver-tarifas-y-detalle.md)** — para comparar precios y condiciones.  
3. **[Carrito y servicios](../general/carrito-y-servicios.md)** — para revisar lo que agregaste.  
4. **[Nominación y finalización](../general/nominacion.md)** — para confirmar y crear la reserva.  
5. **[File](../general/file.md)** — donde gestionás todo después de reservar.

---

:::tip
💡 Podés agregar varios servicios distintos al mismo carrito antes de nominar.  
Esto te permite armar una reserva completa combinando alojamiento, traslados, excursiones o asistencias.
:::
