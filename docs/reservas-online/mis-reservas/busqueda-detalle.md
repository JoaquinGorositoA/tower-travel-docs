- Seguidamente el vendedor deberá completar algunos campos requeridos y hacer clic en el botón *buscar*.

Se podrá hacer la búsqueda por:

- Número de File  
- Cliente  
- Fecha In (Desde, Hasta)  
- Fecha Alta (Desde, Hasta)
